Note that you need the path to the "run" directory.
To get the instructions for any benchmark go to the run directory of interest and run the specinvoke command that runspec runs, with an added -n at the end.

Example:

1) runspec shows: /home/prodromou/Desktop/SPEC_2006/bin/specinvoke -d /home/prodromou/Desktop/SPEC_2006/benchspec/CPU2006/400.perlbench/run/run_base_ref_amd64-m64-gcc43-nn.0005 -e speccmds.err -o speccmds.stdout -f speccmds.cmd -C

2)Switch to the run directory and execute the same instruction, with a -n at the end (two -n if you want the cd instructions as well)

3) The output shows the instructions:
